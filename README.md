## 🎯 Giới thiệu

> 💡 **"Khi AI gặp nghệ thuật lập trình, điều kỳ diệu xảy ra ngay tức thì!"**

**B.Trọng AI** là sân chơi sáng tạo dành cho những tâm hồn yêu công nghệ và nghệ thuật số. Tại đây, bạn sẽ:
- **Giao lưu với AI** để tạo ra những dòng code sống động, biến những ý tưởng thành hiện thực.
- **Khám phá hai chế độ sáng tạo độc đáo:**  
  - `p5.js Mode` cho nghệ thuật thị giác đầy mê hoặc.  
  - `Siêu Đẹp Mode` dành cho giao diện web lung linh và hiện đại.
- **Học hỏi không ngừng:** Mỗi tương tác là một bài học, mỗi dòng code là một tác phẩm nghệ thuật.

---

## 🚀 Tính năng nổi bật

- **💬 Trò chuyện với AI:** Giao tiếp tự nhiên, giống như có một người bạn đồng hành luôn sẵn sàng giúp đỡ.
- **🎨 Hai chế độ sáng tạo:**  
  - `p5.js Mode` – Khơi gợi cảm hứng qua tranh số, hiệu ứng fractal, và những trò chơi vui nhộn.
  - `Siêu Đẹp Mode` – Tạo ra các giao diện web ấn tượng, nơi nghệ thuật và công nghệ hòa quyện.
- **🧠 AI phân tích & gợi ý tự động:** Code của bạn sẽ không bao giờ đơn đi; AI luôn sẵn sàng chêm chút “bí kíp” để nâng tầm sản phẩm.
- **⚡ Hiệu ứng thời gian thực:** Xem kết quả ngay tức thì, cảm nhận sức mạnh của mỗi dòng lệnh bạn viết.
- **🔄 Chuyển đổi cấu hình cực nhanh:** Điều chỉnh các tham số như `temperature`, `top-K`, `top-P` một cách linh động và trực quan.
- **📱 Thiết kế responsive:** Tận hưởng trải nghiệm mượt mà trên mọi thiết bị – từ desktop đến mobile.
- **🔐 Bảo mật tuyệt đối:** Quản lý API Key an toàn qua file `.env`, giúp bạn yên tâm sáng tạo.

---

## 🧭 Chế độ hoạt động

### 🎨 Chế độ 1: `p5.js Mode` – Nghệ thuật tương tác

- **Ý tưởng:** Thả hồn vào các dòng code, vẽ nên bức tranh số độc đáo từ hiệu ứng fractal đến hoạt hình vũ trụ.
- **Trải nghiệm:** Mỗi lần chạy code là một lần mở ra cánh cửa sáng tạo mới.  
- **Ví dụ mẫu:**  
  ```js
  // Hãy cùng tạo bầu trời đầy sao lung linh!
  ellipse(random(width), random(height), 2, 2);
  ```

---

### 💎 Chế độ 2: `Siêu Đẹp Mode` – UI/UX sáng tạo

- **Ý tưởng:** Biến ý tưởng của bạn thành những trang web sống động, nơi hiệu ứng 3D và CSS hiện đại hòa quyện.
- **Trải nghiệm:** Tạo landing page, portfolio hay landing ấn tượng chỉ với vài dòng code.  
- **Công nghệ:** Sức mạnh của **HTML**, **Tailwind CSS**, **GSAP** và **Three.js** được tích hợp hoàn hảo.

## 🛠️ Công cụ Nhà phát triển (Beta)

> ⚠️ **Bí mật dành cho những người đam mê sáng tạo:** Nhấn 7 lần vào tiêu đề “B.Trọng AI” để khám phá menu nhà phát triển – nơi chứa đựng rất nhiều tính năng “cool ngầu”!

### ⚙️ Cấu hình & Tùy chọn

- **Lựa chọn model AI:** `Gemini`, `OpenAI`, `Claude`, `DeepSeek`, và nhiều hơn thế!
- **Điều chỉnh tham số:** Tinh chỉnh `temperature`, `top-K`, `top-P`, `max tokens` để phù hợp với từng ý tưởng.
- **Tính năng nổi bật:**
  - 🔄 **AI Morph & Evolve:** AI tự động gợi ý nâng cấp code, giúp bạn không bao giờ dừng lại ở cùng chỗ.
  - 🛡️ **Rối mã JS:** Bảo vệ tác phẩm của bạn với 4 cấp độ bảo vệ chống sao chép.
  - 🎤 **Điều khiển giọng nói:** Tương tác hands-free – nói, và AI sẽ phản hồi ngay lập tức.
  - 🖼️ **Sinh ảnh từ văn bản:** Từ mô tả đến bức tranh – AI biến ý tưởng thành hình ảnh sống động.
  - 🧽 **Xóa ngữ cảnh:** Làm mới toàn bộ trạng thái của AI chỉ với một click, để ý tưởng luôn bùng nổ.

## 📦 Cài đặt & Sử dụng

1. **Clone dự án:**

   ```bash
   git clone https://github.com/Trongdepzai-dev/AI-Making-Web.git
   ```

2. **Di chuyển vào thư mục dự án:**

   ```bash
   cd btrong-ai
   ```

3. **Cài đặt các phụ thuộc:**

   ```bash
   npm install
   ```

4. **Khởi chạy ứng dụng:**

   ```bash
   npm run dev
   ```

🔗 **Truy cập ngay:** [http://localhost:5173](http://localhost:5173)

## 🔐 Cấu hình API Key

File cấu hình: **`.env` hoặc `.local`**

```env
API_KEY=your_google_genai_key
OPENAI_API_KEY=your_openai_key
CLAUDE_API_KEY=your_anthropic_key
LLAMA_API_KEY=your_meta_llama_key
```

> **Chú ý:** Giữ kín thông tin API, đừng để lộ file cấu hình này trên mạng nhé!
